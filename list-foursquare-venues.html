List Foursquare Venues

Foursquare is insanely popular these days. The venues project developer site outlines code to GET a json object containing values in brooklyn. The request is like so (when using curl):

curl "https://api.foursquare.com/v2/venues/search?ll=40.7,-74&client_id=CLIENT_ID&client_secret=CLIENT_SECRET"

Sign up for a foursquare developer oauth account and print out a list of the venus returned using the url specified above.

Use jQuery's $.ajax to get the data, and attach the print function to the button below.
